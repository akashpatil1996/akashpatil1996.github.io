Face Mask Detector

Face Mask Detector is an AI-based application which detects whether a person is wearing a face mask or not in a given image. The application is developed using Convolutional Neural Networks (CNN) and trained on a dataset of images with and without masks.

To use the Face Mask Detector web app, simply visit the website and upload a photo. The application will analyze the image and provide a result indicating whether a mask is being worn or not.
